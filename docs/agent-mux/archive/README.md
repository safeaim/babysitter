# agent-mux archive

This directory preserves historical and exploratory material related to `@a5c-ai/agent-mux`.

Nothing in this directory is the source of truth for current behavior. Use it for context, comparisons, research history, and design archaeology.

## Archive tracks

- [Comparisons](comparisons/README.md) — parity analyses, external reference surveys, capability matrices
- [Design](design/README.md) — launcher, provider-mux, and other proposal-stage or historical design documents
- [Research](research/README.md) — exploratory investigations and implementation planning notes
- [Superpowers](superpowers/README.md) — historical plan/spec artifacts kept for traceability

## Usage rule

When archived material conflicts with `../reference/`, the reference docs win.
