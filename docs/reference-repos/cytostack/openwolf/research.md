## Summary
Middleware that enhances Claude Code with project intelligence, token tracking, and learning memory through 6 hook scripts. Features file indexing via `anatomy.md` with token estimates, learning memory through `cerebrum.md` capturing preferences and corrections, token ledger tracking lifetime usage across 132+ sessions with 65.8% average reduction, and bug memory preventing rediscovery of known issues. Autonomous middleware that manages context without workflow changes.
