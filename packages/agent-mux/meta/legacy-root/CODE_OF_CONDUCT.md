# Contributor Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

- Be respectful. Assume good faith.
- No harassment, discrimination, or personal attacks.
- Disagree on ideas, not people.
- Report incidents privately to the maintainers (see SECURITY.md for contact).

Enforcement follows the Contributor Covenant's enforcement guidelines.
