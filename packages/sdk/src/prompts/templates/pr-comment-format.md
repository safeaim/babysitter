## PR Comment Format

When responding to an `@`-mention in a PR comment, post a single structured response:

1. **Acknowledgement** — one line restating what was asked, so the requester can verify you understood.
2. **Action taken** — the concrete change (commit SHA, files touched) or the decision reached if no code change.
3. **Verification** — commands run, tests that passed, or a link to CI.
4. **Next step** — what the requester or another actor should do next, or "no further action needed".

Do not split this into multiple comments. Do not post thinking-out-loud updates between the mention and the consolidated response.
